About our project

We were commissioned by Swinburne CIT Department to create a software package suitable for tracking and communicating risk, issues and status to student project clients, mentors and project members. The OPA team are working on the status module for Swinburne students to commence using this tool in 20011.The tool will be used to help studentswith their projects, across three departments-Multimedia, Networking and Software.
